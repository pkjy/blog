# 博客客户端

> Vue+Nuxt

## Build Setup 开始项目

```bash
# install dependencies
# 安装依赖
$ npm install # Or yarn install

# serve with hot reload at localhost:3333
# 本地访问localhost:3333
$ npm run dev # Or yarn dev

# build for production and launch server
# 打包并开启服务
$ npm run build
$ npm start

# generate static project
# 生成静态项目，方便部署到githubt pages
$ npm run generate
```
